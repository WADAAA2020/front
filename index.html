Comandos básicos

Crie um script que pergunte o nome do usuário, e depois exiba uma mensagem na tela "Olá, {nome do usuário}".
Crie um script que pergunte a disciplina favorita ao usuário, e depois exiba uma mensagem na tela "A matéria {materia} realmente é ótima".
Crie um script que pergunte o dia que é hoje para o usuário (ex: 20/02/2002) e exiba na tela "Hoje é dia {data}".
Crie um script que pergunte para o usuário "Está preparado para o segundo semestre de Desenvolvimento de Sistemas? (sim/não)".

Operadores aritméticos

Crie um script que peça ao usuário para inserir dois números. Calcule e exiba o resultado das quatro operações aritméticas (soma, subtração, multiplicação e divisão) entre esses dois números.
Crie um script que peça 3 notas e depois mostre a média.
Crie um script que pergunte a a largura e a altura de uma parede. Sabendo que 1 litro de tinta pinta 2 metros quadrados, calcule quantos litros de tinta serão necessários.
Crie um script que calcule a diferença de 2 números.
Crie um script que peça 2 números e verifique se o 1º é múltiplo do 2º. (ex.: 9 é múltiplo de 3).
Crie um script que converta uma temperatura de Celsius para Fahrenheit (°F = °C x 9/5 + 32).
Crie um script que calcule a área (base * altura).

Condicionais

Crie um script que peça a idade e exiba se a pessoa é maior de idade ou não.
Crie um script que peça 1 número e exiba se é par ou ímpar.
Criar um script que peça 2 números e exiba se o 1º ou 2º é maior ou se eles são iguais.
Crie um script que verifica se a senha está correta ou não. dica a senha é jsélegal
Crie um script que peça para o usuário informar a temperatura em graus Celsius. ele ira falar se o clima está frio(< 18), agradável(> = 18 e <=25) ou quente.

Exercícios de fixação

Crie um script que peça 2 notas e informe se está acima da média, e se é uma nota par, e mostre o resultado ao usuário.
Crie um script que peça o peso e a altura e mostre o IMC do usuário.
Crie um script que peça para o usuário um ano e informe se ele é bissexto.
Crie um script que peça um valor de moeda(Dólar(1), Euro(2), Won sul-coreano(3) e Iene(4)), e um valor desejado, exiba quanto ficaria esse valor com a taxa.

Laços de repetição

Crie um script que peça para o usuário informar um número de 1 a 9, mostrar a tabuada do número escolhido do 0 ao 10.
Crie um script do jogo pedra papel tesoura, o computador sempre vai ser pedra, informar o resultado.
Crie um script que peça um número para o usuário e calcule o fatorial.
